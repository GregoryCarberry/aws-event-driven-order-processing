
Architecture Notes
MVP Architecture

The first version will use API Gateway, Lambda, DynamoDB, and DynamoDB Streams.

The aim is to keep the architecture simple enough to explain clearly, while still showing useful cloud engineering concepts such as asynchronous processing, state changes, and logs.
